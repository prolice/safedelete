Installation

To use the SafeDelete app, follow these steps:

  1. Download the latest version of the app from the repository.
  2. Install the required dependencies.
  3. Set up the app configuration.
  4. Run the app on your local machine or deploy it to a server.

Usage

To use the SafeDelete app, follow these steps:

  1. Launch the app on your device.
  2. Select the files or folders you want to delete securely.
  3. Click on the "Safe Delete" button to securely delete the selected files.

Classes

SafeDelete

The SafeDelete class provides methods for securely deleting files and folders.

  • constructor(options) - Creates a new instance of the SafeDelete class with the specified options.
  • deleteFile(filePath) - Securely deletes the file at the specified file path.
  • deleteFolder(folderPath) - Securely deletes the folder and its contents at the specified folder path.

Functions

encryptFile(filePath)

This function encrypts the file at the specified file path using a secure encryption algorithm.

  • filePath - The path to the file to be encrypted.

overwriteWithRandomData(filePath)

This function overwrites the contents of the file at the specified file path with random data to securely delete its original contents.

  • filePath - The path to the file to be overwritten with random data.

Examples

Here's an example demonstrating the usage of the SafeDelete app: 

// Create an instance of SafeDelete
const safeDelete = new SafeDelete();

// Delete a file securely
safeDelete.deleteFile('/path/to/file.txt');

// Delete a folder securely
safeDelete.deleteFolder('/path/to/folder');
