SecureFileDelete User Manual

Introduction

SecureFileDelete is a command-line tool that securely deletes files from a specified folder while ensuring that the deleted files cannot be easily recovered. This tool overwrites the file content with multiple iterations of random data, null values, zero values, alternating patterns, and encrypted random data using various steps and algorithms.

System Requirements

To use the SecureFileDelete tool, make sure your system meets the following requirements:

  • Operating System: Windows
  • .NET Framework: Version 4.7 or above

Installation

The SecureFileDelete tool does not require any installation process. You can directly use the provided C# code to compile and run the tool.

Usage

To use the SecureFileDelete tool, follow the instructions below:

  1. Open the command prompt or terminal on your system.
  2. Navigate to the directory where the SecureFileDelete C# code file is located.
  3. Compile the code using the C# compiler. Run the following command:
csc SecureFileDelete.cs

This will generate an executable file named SecureFileDelete.exe.

  1. Run the SecureFileDelete tool by executing the following command:
SecureFileDelete.exe --folder <folder_path> [--time_limit <minutes>] [--pattern <file_pattern>] [--exclude_files <file1,file2,...>]

Command-line Parameters

The SecureFileDelete tool accepts the following command-line parameters:

  • --folder: Specifies the path of the folder from which files will be deleted. This parameter is required.
  • --time_limit: Specifies the time limit (in minutes) that defines the maximum age of files to be deleted. Only files created within the specified time limit will be deleted. This parameter is optional.
  • --pattern: Specifies the file name pattern to match. Only files with names containing the specified pattern will be deleted. This parameter is optional.
  • --exclude_files: Specifies a comma-separated list of files to exclude from deletion. These files will be ignored during the secure file deletion process. This parameter is optional.

Examples

Here are some examples demonstrating the usage of the SecureFileDelete tool:

SecureFileDelete.exe --folder C:\Data --time_limit 60 --pattern *.txt --exclude_files file1.txt,file2.txt

Important Notes

  • Ensure that you have appropriate permissions to delete files from the specified folder.
  • Use the tool responsibly and exercise caution when permanently deleting files. Once files are securely deleted, they cannot be recovered.
