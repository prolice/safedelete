SecureFileDelete User Manual

Introduction

Welcome to the SecureFileDelete user manual. SecureFileDelete is a command-line tool designed to securely delete files from a specified folder. This manual provides step-by-step instructions on how to use the SecureFileDelete application.

Getting Started

To get started with SecureFileDelete, follow the steps below:

  1. Ensure you have the latest version of SecureFileDelete installed on your system.
  2. Open a command prompt or terminal.
  3. Navigate to the directory where the SecureFileDelete executable is located.

Usage

The SecureFileDelete application supports the following command-line parameters: 


        --folder, -f          : The path of the folder to delete files from. (required)
        --time_limit, -t      : The time limit (in minutes) specifying the maximum file age. (optional)
        --pattern, -p         : The file name pattern to match. (optional)
        --exclude_files, -e   : Comma-separated list of files to exclude from deletion. (optional)
        --recursive, -r       : Search for files in all subfolders. (optional)
        --no_prompt, -np      : Enable no prompt mode. Files will be securely deleted without confirmation. (optional)
        --simulate, -s        : Enable simulate mode. Files will not be securely deleted, they will be simulated. (optional)
        --help, -h            : Display usage information. (optional)

Examples

Here are some examples of how to use the SecureFileDelete application:


        safedelete --folder "C:\Documents" --time_limit 30 --pattern "*.txt" --exclude_files "important.txt" --recursive
        safedelete -f "C:\Documents" -t 60 -p "*.docx" -e "backup.docx,temp.docx" -r -np

Important Notes

  • Ensure that you have the necessary permissions to access and delete files from the specified folder.
  • Be cautious when using the --no_prompt or --simulate options, as they can lead to unintended file deletion.

Troubleshooting

If you encounter any issues or have questions regarding the SecureFileDelete application, please refer to the official documentation or contact our support team for assistance.

Conclusion

Congratulations! You have successfully learned how to use the SecureFileDelete application to securely delete files from a specified folder. Make sure to use the application responsibly and exercise caution when deleting files.
