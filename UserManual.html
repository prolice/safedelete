User Manual - SafeDelete

Overview

The SafeDelete program is used to securely delete files from a specified folder based on various criteria.

Usage

Usage: safedelete --folder <folder_path> [--time_limit <minutes>] [--pattern <file_pattern>] [--exclude_files <file1,file2,...>] [--recursive]
       safedelete -f <folder_path> [-t <minutes>] [-p <file_pattern>] [-e <file1,file2,...>] [-r]
    
Parameters:
  --folder, -f          : The path of the folder to delete files from. (required)
  --time_limit, -t      : The time limit (in minutes) specifying the maximum file age. (optional)
  --pattern, -p         : The file name pattern to match. (optional)
  --exclude_files, -e   : Comma-separated list of files to exclude from deletion. (optional)
  --recursive, -r       : Search for files in all subfolders. (optional)
  --no_prompt, -np      : Enable no prompt mode. Files will be securely deleted without confirmation. (optional)
  --help, -h            : Display usage information. (optional)

Examples


        safedelete --folder C:\Temp
        safedelete -f C:\Temp -t 60 -p *.txt.   --exclude_files file1.txt,file2.txt -r
        safedelete --folder /home/user/files --time_limit 30 --exclude_files important.docx,personal.jpg --recursive

Notes

  • If the `--time_limit` parameter is not specified, all files in the folder will be considered for deletion.
  • The `--pattern` parameter accepts wildcards (*) to match multiple files. For example, `*.txt` will match all text files.
  • When using the `--exclude_files` parameter, provide a comma-separated list of files to exclude from deletion.
  • When the `--recursive` parameter is used, SafeDelete will search for files in all subfolders of the specified folder.
  • By default, SafeDelete prompts for confirmation before deleting each file. Use the `--no_prompt` parameter to enable no prompt mode.

Security Considerations

SafeDelete uses secure deletion techniques to prevent file recovery. However, ensure that you have the necessary permissions to delete files from the specified folder.
